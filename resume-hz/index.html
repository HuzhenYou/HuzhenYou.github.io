Backtrader 股票策略回测

独立完成· 独立研发 · 已上线

AI 赋能的无代码回测系统：自然语言描述策略 → LLM 生成可执行策略公式 → 确定性校验闭环 → 回测信号可视化。

  • 可验证生成闭环（RLVR 思想工程化）：公式生成后在 10 只样本股上 AST parse + 求值 dry-run，失败原因回注 prompt 重采样（≤3 次），全部通过才交付；AST 审计自动检测「使用未来信息」的前瞻函数（防 repaint 作弊）
  • 运行时数学不变量判分：16 条硬不变量（概率 ∈ [0,1]、胜负计数守恒等）违反即抛错，拒绝静默填充掩盖
  • 为保障计算正确性，设计多 agent 对抗审计工作流：独立审计员分角度扫描 + 对抗验证器三路证伪（机制 / 数值 / 设计意图），以「连续两轮零发现」为收敛判据；25 轮累计 132+ 角度，误报受控（单轮 8 confirmed / 0 rejected），回归锁点 253 → 383 条并以「注入旧实现验证锁点变红」确认测试有效
  • 行情数据 pipeline：4598 只 A 股多源下载 + 前复权口径对齐 + 覆盖率完整性自检（区分真停牌与下载失败）+ 黄金用例对账
